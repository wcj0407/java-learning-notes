# 字符流

## 1.特点

字符流=字节流＋字符集

特点
输入流:一次读一个字节，遇到中文时，一次读多个字节
输出流:底层会把数据按照指定的编码方式进行编码，变成字节再写到文件中

## 2.FileReader

```
public class demo2 {
    public static void main(String[] args) throws IOException {
        FileReader fr = new FileReader("D:\\zhuomian\\IOliu\\a.txt");

        int a;
        while((a = fr.read())!=-1) {
            System.out.print((char)a);
        }
        fr.close();


    }
}
```
