# 文件拷贝



```
public class democopy {
    public static void main(String[] args) throws IOException {
        FileInputStream fis = new FileInputStream("D:\\zhuomian\\aaa\\aaa.png");
        FileOutputStream fos = new FileOutputStream("D:\\zhuomian\\aaa\\bbb.png");
        int b;
        while((b=fis.read())!=-1) {
            fos.write(b);
        }

        fos.close();
        fis.close();

    }
}
```

### 文件拷贝就是利用文件的一遍读一遍写的方式来复制

#### 假如要是一个一个字节复制那太慢了，可以用一个一个数组来加快速度

#### 如：

```
public class democopy2 {
    static long start =System.currentTimeMillis();
    public static void main(String[] args) throws IOException {
       FileInputStream fis = new FileInputStream("D:\\zhuomian\\aaa\\7.mp4");
        FileOutputStream fos = new FileOutputStream("D:\\zhuomian\\aaa\\7-7.mp4");

        byte [] bytes =new byte[1024*1024*5];
        int len =0;
        while((len =fis.read())!=-1) {
            fos.write(bytes,0,len);

        }
        fos.close();
        fis.close();

        long end = System.currentTimeMillis();
        System.out.println(end-start);
    }
}
```
