# 可变参数

格式：

数据类型...数组名

```
public class getsum {

    public static void main(String[] args) {
        int sum=getsum(1,2,3);
        System.out.println(sum);

    }


    public static int getsum(int...a) {
        int sum = 0;
       for (int i :a) {
           sum += i;
       }
       return sum;
    }
}
```
