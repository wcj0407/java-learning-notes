# 对遍历知识进行回忆加巩固

### 键找值（KeySet遍历）

① Map中获取所有键的方法叫什么？

叫keySet()方法

② 这个方法返回的是什么类型的集合？

返回的是Set<K>比如Set<String>类型的集合

③ 拿到每一个键之后，通过什么方法获取对应的值？

通过map.get(key)方法

④ 如果Map中不存在这个键，get方法返回什么？

null



### 2. 键值对遍历（EntrySet遍历）

① Map中获取所有键值对对象的方法叫什么？

map.entrySet()

② 返回的集合元素类型是什么？

Set<Map.Entry<K,V>>比如Set<map.entry<String,string>>

③ Entry对象获取键的方法叫什么？

entry.getkey

④ Entry对象获取值的方法叫什么？

entry.getvalue

⑤ 键值对遍历和键找值遍历相比，哪个效率更高？为什么？

键值对遍历更高，因为当键值对遍历的时候已经把全部数据拿到了，要是键找值得话，第一次那到key之后还得再遍历一次拿到值



### 3. Lambda遍历

① Map中的Lambda遍历使用什么方法？

map.forEach((k, v) -> { })

用得是map.foreach(s)->{}）

② forEach方法需要传递什么参数？

BiConsumer<K, V>

③ Lambda表达式本质上是简化了什么东西？

简化内部类

④ Lambda中的两个参数分别表示什么？

key,value

# 第二部分：HashMap基本使用

### 4. HashMap特点

① HashMap存储的是哪两部分数据？

存储的是键和值

② HashMap中的键可以重复吗？

键不可以重复，

③ HashMap中的值可以重复吗？

值可以重复

④ HashMap中的键和值允许为null吗？

允许1个 null key

多个 null value

### 5. HashMap常用方法

① 添加元素的方法是什么？

map.put()

② 删除键对应元素的方法是什么？

map.remove()

③ 根据键获取值的方法是什么？

map.get(key)

④ 判断是否包含某个键的方法是什么？

map.containsKey(key)

⑤ 获取集合长度的方法是什么？

map.size()

# 第三部分：HashMap存储自定义对象

### 6. 自定义对象作为键

① 如果Student对象作为HashMap的键，想让内容相同的对象视为同一个键，需要重写哪两个方法？

在类中重写hashcode和equals方法

② 为什么只重写equals不够？

因为只重写equals，在放数据之前你得先知道数据应该存那个位置啊，equals方法只是比较数据有没有重复，和重复了之后应该怎么操作而已,hashcode方法是第一步要进行操作的

③ HashMap判断两个键相同，先比较什么？再比较什么？

1. hashCode

2. equals
